# Plugins

> **Reserved** — This guide is pending the shell/app/plugin platform plan. It will cover developing and distributing plugins that extend Mog with custom functions, views, data connectors, and UI panels.

When approved, this guide will document:

- Plugin manifest format and lifecycle hooks
- Custom formula function registration
- Custom view type registration (extending the Views layer)
- Data connector plugins (external data sources)
- UI panel plugins (sidebar, dialog, toolbar extensions)
- Plugin sandboxing and capability grants
- Distribution via plugin registry

See [Architecture Overview](architecture-overview.md) for the layer model that plugins extend.
